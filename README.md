Apache Log File Analysis
===============
Analyses an apache log file and produces :
* Total Megabytes Sent
* Mean Response Time
* Successful Requests per Minute
* Error Requests Per Minutes

Requirements
------------
 - Apache Web Access Log

Usage
-----
```[user@computer ~]# ./parse.rb access.log
